Basic styling attribute to start working with a flexbox

Basic cmds for creating a good flex-box

flex-wrap : wrap

Use flex-wrap to wrap the items in the container

flex-basis

Basically, same as min-width attribute the major difference is that it doesn't show a scroll bar when you adjust the window size

flex:[-grow][-shrink][-basis]

By default basis is set to 0,given one value; the same value will be taken by grow and shrink

justify-content

Space-between, flex-start, flex-end, space-around are a few set of attributes that can be used to style the content position
